# Facial_Expression_project
Facial Expression Recognition using CNN and Deep Learning# 😃 Facial Expression Recognition Project

## 📌 Project Description
This project is a Deep Learning based Facial Expression Recognition system. It detects human emotions from images/webcam using a CNN (Convolutional Neural Network).

---

## 🚀 Features
- Detects facial expressions in real-time
- Uses CNN model for classification
- Can identify emotions like:
  - Happy 😊
  - Sad 😢
  - Angry 😡
  - Surprise 😲
  - Neutral 😐

---

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- CNN (Deep Learning)
